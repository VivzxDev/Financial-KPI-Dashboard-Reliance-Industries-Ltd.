🏢 Project 1 — Financial KPI Dashboard
Reliance Industries Ltd.

💡 “Turning financial statements into strategic insights.”

🔍 What I did:

Analyzed 3 years of financial data (FY23–FY25) to calculate 9+ critical KPIs including:
📈 Revenue Growth (9.97%)
💰 EBITDA Margin
📊 Net Profit Margin
📉 Return on Equity (ROE) & Return on Assets (ROA)
⚖️ Debt-to-Equity Ratio

📉 Key Insights:

Identified a consistent decline in EBITDA Margin, signaling potential operational inefficiencies.
Observed a deteriorating ROE trend, raising concerns around capital allocation effectiveness.
Highlighted areas where management attention could improve profitability and efficiency.

📊 Dashboard Highlights:

Built an interactive Power BI dashboard featuring:
KPI Cards ⚡
Trend Lines 📈
Year-on-Year Comparison Charts 🔄
Designed for quick executive decision-making and financial health assessment.

🧠 Techniques Used:

Vertical & Horizontal Analysis
Ratio Benchmarking
Financial Statement Interpretation

🛠️ Tools:
Excel | Power BI
